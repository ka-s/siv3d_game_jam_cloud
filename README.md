# Spider was born from the cloud 〜雲から生まれた蜘蛛〜

第2回Siv3D Game Jam
テーマ「雲」

Create by かーず

![スクリーンショット1](/ss2.jpg)

# ゲーム説明

雲から蜘蛛が落ちてきて拠点を襲ってきたので守ろう！

# 操作方法

数字の1キーで兵士召喚(コスト10払う)(コストは左上に記載)
Zキーで攻撃

# 解説

いわゆるタワーディフェンスみたいなゲームです。
割りと時間が取れず実装したいところが実装できませんでしたが、
結果的にカオスなげーむが出来上がったので結果オーライです。

コードに関しては最後に2日間くらいはババっと書いたのでアレですが、
それ以前は頑張って綺麗に書こうとしてました。(綺麗とは言っていない)

とりあえず楽しかったです！
